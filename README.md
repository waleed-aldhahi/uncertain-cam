## Installation
```
git clone https://github.com/waleed-aldhahi/uncertain-cam.git
cd uncertain-cam
pip install -r requirements.txt  # pip install -e .
python run.py
```
## Or
```
pip install git+https://github.com/waleed-aldhahi/uncertain-cam.git
```
## Run 
```
python run.py
```
